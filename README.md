# Drupal Blog Front
---

### Installation 🚀

`npm install`
###### Start Project 🏁
`npm start`

---

### Pages 📄
- Home Page: `/` 
- Blogs Listing Page: `/blogs`
- Blog Page: `/blog/{{id}}`
- Blogs By Topic Page: `/topic/{{name}}`
- Bookmarked Blogs Page: `/bookmarks`
- Admin Login Page: `/admin`
- Add Blog Page: `/admin/add-blog`
- Logout: `/logout`
